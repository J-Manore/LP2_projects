# 🎟️ MERN Event Registration Deployment (EXAM VERSION — SIMPLE + REAL ISSUES)

---

## 🚀 FLOW (just remember this)
```
SSH → setup → clone → backend → frontend → run
```

---

## 1. SSH
```bash
ssh -i ~/key.pem azureuser@<IP>
```

---

## 2. Setup
```bash
sudo apt update && sudo apt install -y git curl build-essential
```

---

## 3. ADD SWAP (VERY IMPORTANT)
```bash
sudo fallocate -l 1G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
```

---

## 4. Install Node (USE 20)
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
```

---

## 5. Install Mongo (FIXED FOR UBUNTU 24.04)
```bash
curl -fsSL https://www.mongodb.org/static/pgp/server-6.0.asc | sudo gpg --dearmor -o /usr/share/keyrings/mongodb-server-6.0.gpg

echo "deb [arch=amd64,arm64 signed-by=/usr/share/keyrings/mongodb-server-6.0.gpg] https://repo.mongodb.org/apt/ubuntu jammy/mongodb-org/6.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-6.0.list

sudo apt update
sudo apt install -y mongodb-org
sudo systemctl start mongod
```

---

## 6. Clone
```bash
git clone <repo>
cd mern-event-registration
```

---

## 7. Backend
```bash
cd backend
npm install

cat <<EOT > .env
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/event-db
EOT

node server.js
```
**Test:**
```bash
curl http://localhost:5000/api/events
```

---

## 8. Frontend
```bash
cd ../frontend
npm install --no-audit --no-fund --legacy-peer-deps
VITE_API_URL=http://<IP>:5000/api/events npm run build
```

---

## 9. Run
```bash
sudo npm install -g pm2 serve

cd ../backend
pm2 start server.js --name "api"

cd ../frontend
pm2 serve dist 3000 --spa --name "ui"
```

---

## 10. Open
```
http://<IP>:3000
```

# 🐛 ALL ISSUES YOU FACED (REAL DEBUG LIST)

---

## 1. SSH KEY ERROR (WSL)
**Error:** `UNPROTECTED PRIVATE KEY FILE`
**Fix:**
```bash
cp /mnt/c/.../key.pem ~/
chmod 400 ~/key.pem
```

---

## 2. Mongo install fails
**Error:** `Unable to locate package mongodb-org`
**Cause:** Ubuntu 24 (noble) not supported properly out of the box.
**Fix:** use `jammy` repo (already fixed in step 5)

---

## 3. npm install ENOENT
**Error:** `package.json not found`
**Cause:** wrong folder
**Fix:** 
```bash
cd backend
# or
cd frontend
```

---

## 4. curl not working
**Error:** `Failed to connect`
**Fix:** Make sure the backend is actually running:
```bash
node server.js
```

---

## 5. Node version issue
**Error:** Vite requires Node 20
**Fix:** Install Node 20 instead of 18 (already fixed in step 4).

---

## 6. Website not opening
**Error:** `ERR_CONNECTION_REFUSED`
**Fix:** 👉 open port **3000** in Azure Portal.

---

## 7. Vite native binding error
**Error:** `Cannot find native binding`
**Fix:**
```bash
rm -rf node_modules package-lock.json
npm install
```

---

## 8. npm install hangs
**Cause:** 1GB RAM on VM.
**Fix:** 
```bash
npm install --no-audit --no-fund --legacy-peer-deps
```
* AND make sure you ADDED SWAP (Step 3).

---

## 9. SSH becomes slow / VM freezes
**Cause:** memory overload.
**Fix:** 👉 restart VM from Azure portal.

---

## 10. VM agent not ready / SSH dead
**Fix:** 👉 delete VM → recreate.

---
