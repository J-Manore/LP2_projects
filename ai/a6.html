Hospital Expert System

AI Based Symptom Checking System

About System

This expert system helps users identify possible health conditions based on symptoms entered by the user.

The system uses a knowledge base and inference engine to provide suggestions.

Knowledge Base

  • Fever → Infection
  • Cough → Cold
  • Headache → Stress or Fatigue
  • Chest Pain → Emergency Condition
  • Stomach Pain → Digestion Problem

Inference Engine

The inference engine compares the user symptoms with rules stored in the knowledge base and generates conclusions.
